# AI Interview Bot Project Report

## Objective
Build a practical full-stack interview preparation platform for semester evaluation that demonstrates frontend, backend, database, and AI integration in a single clean project.

## Modules
- Authentication
- Dashboard analytics
- Mock interview session
- AI question generation
- AI answer evaluation
- Performance analytics
- Question bank
- Profile management
- Feedback report system

## Design Goals
- Modern SaaS dashboard appearance
- Purple and white visual identity
- Easy viva explanation
- Modular and readable codebase
- Local AI via Ollama for offline demo capability

## Database Summary
The PostgreSQL schema stores users, interviews, questions, responses, and reports. This gives the app persistent history and makes analytics possible.

## Why This Project Is Strong for Review
- Clear full-stack separation
- Real authentication
- Real database queries
- Practical AI use with local models
- Useful charts and interview tracking
- Easy to demonstrate live in class
